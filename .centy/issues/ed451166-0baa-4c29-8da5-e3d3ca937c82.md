# Remove per-entity gRPC RPCs in favor of generic ones

Parent: #179
Depends on: #190

Remove the old per-entity RPCs (CreateIssue, GetDoc, UpdatePr, etc.) now that the generic CRUD RPCs are in place.

## Scope

* Remove all per-entity proto RPC definitions (CreateIssue, GetIssue, ListIssues, UpdateIssue, DeleteIssue, etc. for each item type)
* Remove per-entity handler implementations
* Remove per-entity proto messages (CreateIssueRequest, GetDocRequest, etc.)
* Update all clients/tests to use the generic RPCs
* Clean up server router registrations
