# Field validation based on config.yaml schema

Parent: #130

Validate item metadata (YAML frontmatter) against the config.yaml schema when creating or updating items.

## Scope

* Validate status field against config.yaml statuses list
* Validate priority against config.yaml priorityLevels
* Validate custom fields match config.yaml customFields definitions
* Return clear error messages: field name, expected values, actual value
