# Create Timestamped and SoftDeletable traits

Parent: #105

Create the two foundational traits in src/common/traits.rs:

````rust
pub trait Timestamped {
    fn created_at(&self) -> &str;
    fn updated_at(&self) -> &str;
    fn touch(&mut self);
}

pub trait SoftDeletable: Timestamped {
    fn deleted_at(&self) -> Option<&str>;
    fn is_deleted(&self) -> bool { self.deleted_at().is_some() }
    fn soft_delete(&mut self);
    fn restore(&mut self);
}
````

## Scope

* Create src/common/traits.rs with both traits
* Export from src/common/mod.rs
* Unit tests for default method is_deleted()
