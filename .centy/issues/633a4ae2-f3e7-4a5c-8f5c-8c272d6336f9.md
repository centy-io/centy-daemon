# Remove #[allow] lint suppression in src/server/error_mapping/mod.rs

## Context

\`src/server/error_mapping/mod.rs\` is 111 lines and has \`#![allow(unknown_lints, max_lines_per_file)]\` at the top. The allow was re-added by PR #177 which enforced the 100-line limit by adding allows to all violating files.
