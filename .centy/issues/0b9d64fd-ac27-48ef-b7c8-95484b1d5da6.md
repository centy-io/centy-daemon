# Sync & git integration: auto-sync, GitHub sync, user data, and gix refactor

This issue consolidates all sync and git integration work into a single tracked effort. It covers the dedicated centy branch sync architecture, GitHub/GitLab issue mirroring, user data completeness during sync, and the underlying git library refactor.

## Background

Centy needs a robust, reliable sync story across multiple dimensions:

1. **Centy branch auto-sync** — a dedicated orphan `centy` branch acts as the "database" for all `.centy/` files, kept in sync (pull + push) on every read/write operation, enabling multi-machine collaboration and backup.
2. **GitHub/GitLab issue sync** — bidirectional (or at minimum import) sync of issues between centy and GitHub/GitLab.
3. **User data completeness** — when syncing git contributors to centy users, existing users with missing data (e.g. empty `git_usernames`) should be updated rather than skipped.
4. **gix refactor** — replace the ~56+ `Command::new("git")` subprocess calls with the pure-Rust `gix` crate for performance, type safety, and testability.

## Architecture Reference (centy branch sync)

```
┌─────────────────────────────────────────────────────────────┐
│                      Daemon (gRPC)                          │
├─────────────────────────────────────────────────────────────┤
│   CreateIssue  GetIssue  UpdateIssue  ...                  │
│       │           │          │                              │
│       └───────────┴──────────┴──────────────────┐          │
│                              ┌───────────────────▼────────┐ │
│                              │     CentySyncManager       │ │
│                              │  • pull_before_read()      │ │
│                              │  • commit_and_push()       │ │
│                              │  • merge_conflicts()       │ │
│                              └───────────────────┬────────┘ │
│                              ┌───────────────────▼────────┐ │
│                              │   Centy Branch Worktree    │ │
│                              │  ~/.centy/sync/{hash}/     │ │
│                              │      └── .centy/           │ │
│                              └───────────────────┬────────┘ │
│                                         │ git push/pull     │
│                              ┌──────────▼─────────────────┐ │
│                              │    Remote: origin/centy    │ │
│                              └────────────────────────────┘ │
└─────────────────────────────────────────────────────────────┘
```

**Sync protocol:**
- Read operation: `git pull` → read file
- Write operation: `git pull` → merge → write → commit → push
