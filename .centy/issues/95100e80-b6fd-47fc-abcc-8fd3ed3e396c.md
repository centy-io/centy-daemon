# Collapse per-entity gRPC RPCs into generic CRUD RPCs

Parent: #130

Replace the 37+ individual gRPC RPCs (CreateIssue, GetDoc, UpdatePr, etc.) with ~7 generic RPCs that accept item_type as a parameter.

## Target RPCs

* CreateItem(item_type, title, description, metadata)
* GetItem(item_type, id)
* ListItems(item_type, filters)
* UpdateItem(item_type, id, updates)
* DeleteItem(item_type, id)
* SoftDeleteItem(item_type, id)
* RestoreItem(item_type, id)

## Scope

* Define new generic proto messages
* Implement generic handlers that dispatch via the item type registry
* Keep old per-entity RPCs as thin wrappers delegating to generic ones (deprecation period)
* Update server router
* Integration tests for generic RPCs
