# Add agent_dylint_lints custom linter

Add the agent_dylint_lints dylint library (https://github.com/tupe12334/agent_dylint_lints) to Cargo.toml workspace metadata so custom lints run via `cargo dylint --all`.

Currently fires warnings for:
- Files exceeding 100 lines
- Functions exceeding 60 lines  
- Function bodies exceeding 100 non-empty lines
- async functions without await
- Wildcard imports
- .expect() usage
