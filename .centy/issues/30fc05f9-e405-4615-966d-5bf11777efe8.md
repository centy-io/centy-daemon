# Generate .mcp.json during init command

When running the `init` command, automatically create a `.mcp.json` file in the project root with the centy MCP server configured.

This file enables Claude Code (and other MCP clients) to discover and use the centy MCP server for the project without any manual setup.

## Expected output

```json
{
  "mcpServers": {
    "centy": {
      "command": "npx",
      "args": ["-y", "centy-mcp"]
    }
  }
}
```

## Notes

- Only create the file if it doesn't already exist (don't overwrite)
- The file should be created alongside the other project init artifacts
