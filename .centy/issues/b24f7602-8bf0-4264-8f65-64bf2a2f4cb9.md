# MCP: centy-mcp incompatible with outdated installed centy-daemon

After fixing the npx cache issue, `npx centy-mcp` starts but immediately exits with:

```
centy-mcp v0.12.1 is incompatible with centy-daemon v0.10.5. Please update centy-mcp.
```

## Root cause

The installed daemon at `~/.centy/bin/centy-daemon` is v0.10.5, while `centy-mcp` v0.12.1 requires a newer version. The error message is misleading — it says "update centy-mcp" but the daemon is what needs updating.

## Fix

Build and install the latest daemon from source:
```bash
cargo build --release
cp target/release/centy-daemon ~/.centy/bin/centy-daemon
```

Then restart the daemon.

## Notes

- The error message wording should be corrected to say "Please update centy-daemon" instead of "Please update centy-mcp"
