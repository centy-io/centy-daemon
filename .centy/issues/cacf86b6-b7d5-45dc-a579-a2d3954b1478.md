# Add generic CRUD gRPC RPCs

Parent: #179

Add new generic gRPC RPCs that accept item_type as a parameter, dispatching to the appropriate handler via the item type registry. The old per-entity RPCs remain unchanged during this step.

## Target RPCs

* CreateItem(item_type, title, description, metadata)
* ListItems(item_type, filters)
* UpdateItem(item_type, id, updates)
* DeleteItem(item_type, id)
* SoftDeleteItem(item_type, id)
* RestoreItem(item_type, id)

## Scope

* Define new generic proto messages and RPCs
* Implement generic handlers that dispatch via the item type registry
* Update server router to register new RPCs
* Integration tests for generic RPCs
