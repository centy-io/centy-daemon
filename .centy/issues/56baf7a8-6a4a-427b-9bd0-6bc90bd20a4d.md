# Extract lint-staged config to its own file

The `lint-staged` configuration is currently inline in `package.json`:

```json
"lint-staged": {
  "*.{rs,md,toml,yml,yaml,json,ts,js,sh,txt}": "cspell --no-progress"
}
```

## Proposed change

Extract the config to a dedicated `.lintstagedrc.js` file and remove the inline entry from `package.json`. This keeps `package.json` lean and makes the lint-staged configuration easier to find and extend.
