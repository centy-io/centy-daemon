# Remove PR entity type

Remove the entire PR (pull request) subsystem from the daemon. PRs will not be a built-in entity type. With the config.yaml approach (#135), the init command creates only issue and doc types. If users need a PR-like type, they can create .centy/prs/config.yaml as a custom item type.
