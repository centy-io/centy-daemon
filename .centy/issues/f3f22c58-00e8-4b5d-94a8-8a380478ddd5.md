# GetItem: fallback to org repo when not found in project

Extend `GetItem` to fall back to the org repo when an item is not found in the project's own `.centy/` (spec: `org-wide-centy-repo`). Depends on #384 (org repo discovery).

## Daemon Logic

1. Look up the item in the project's own `.centy/` as normal.
2. If not found and an org repo is tracked for this project's org:
   - Look up the item in the org repo.
   - If found, return it.
3. If still not found, return the usual not-found error.

## Acceptance Criteria

- [ ] `GetItem` returns an org-wide item when it exists in the org repo but not the project
- [ ] If no org repo is tracked, behavior is unchanged (not-found error as before)
- [ ] Project-local item takes precedence if the same ID somehow exists in both
- [ ] Tests cover: found in project, found in org repo, not found in either, no org repo tracked
