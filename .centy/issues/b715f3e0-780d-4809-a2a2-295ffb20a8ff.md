# Split reconciliation plan/mod.rs — separate plan building, file discovery, and hashing

## Problem

`src/reconciliation/plan/mod.rs` (97 lines) conflates:
- Plan building (comparing templates vs. disk state)
- File hash computation
- User file discovery and categorization
- Hash comparison logic

## Proposal

Split into:
- `plan/mod.rs` — public API / entry point
- `plan/builder.rs` — `build_reconciliation_plan` orchestration
- `plan/managed_files.rs` — handling managed template files
- `plan/user_files.rs` — discovering and categorizing user files
- `plan/hashing.rs` — hash comparison utilities
