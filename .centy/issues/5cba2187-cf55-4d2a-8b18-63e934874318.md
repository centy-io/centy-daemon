# Implement Claude Code plugin marketplace for centy-daemon

## Overview

Create a Claude Code plugin marketplace in this repo so users can install it via:

```
/plugin marketplace add centy-io/centy-daemon
```

This exposes skills for setting up the project and interacting with the Centy MCP server.

## Goals

- Allow any user to onboard quickly by running a single `/plugin marketplace add` command
- Provide skills that guide installation, configuration, and day-to-day MCP usage

## Tasks

### Marketplace scaffold
- [ ] Create `.claude-plugin/marketplace.json` at repo root
- [ ] Define the `centy` plugin entry pointing to `./plugins/centy`

### Plugin manifest
- [ ] Create `plugins/centy/.claude-plugin/plugin.json`

### Skills to implement
- [ ] **`install`** — walks the user through cloning the repo, installing dependencies, building the daemon, and registering it as an MCP server in Claude Code
- [ ] **`mcp-usage`** (or similar) — reference skill describing available MCP tools, how to start/stop the daemon, and common workflows (creating items, syncing, etc.)

### Optional
- [ ] Add `.mcp.json` inside the plugin so installing the plugin also wires up the MCP server config automatically
- [ ] CI validation workflow (`.github/workflows/validate-plugins.yml`)

## Acceptance Criteria

- `plugins/centy/skills/install/SKILL.md` guides a new user from zero to a working local daemon connected to Claude Code
- `plugins/centy/skills/mcp-usage/SKILL.md` documents the key MCP operations so Claude can assist with Centy tasks out of the box
- Users can discover and install via `/plugin marketplace add centy-io/centy-daemon`
