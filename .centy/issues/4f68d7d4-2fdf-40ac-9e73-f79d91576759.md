# Split managed_files_merge.rs — separate line-based and JSON merge strategies

## Problem

`src/reconciliation/managed_files_merge.rs` (191 lines) contains two fundamentally different merge strategies in one file:
- Line-based content merging (`LineEnsureMerge`)
- JSON merging via RFC 7396 Merge Patch
- String array union logic shared between them

## Proposal

Extract into a `merge/` submodule:
- `merge/mod.rs` — shared types / entry points
- `merge/lines.rs` — `LineEnsureMerge` logic
- `merge/json.rs` — JSON merge patch logic
- `merge/arrays.rs` — array union utilities
